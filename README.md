API Endpoint Documentation Authentication POST /api/register Register a new user. Body { "username": "user123", "password": "password" } POST /api/login Login and receive JWT. Body { "username": "user123", "password": "password" } Response { "token": "JWT_TOKEN" } Use token in all protected routes: Authorization: Bearer <token> Tasks GET /api/tasks Returns all tasks for authenticated user.
